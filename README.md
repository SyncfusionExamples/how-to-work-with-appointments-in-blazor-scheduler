# How to Work with Appointments in Blazor Scheduler    

## Repository Description

This sample demonstrates comprehensive appointment management capabilities in Syncfusion's Blazor Scheduler component, showcasing various appointment options, configurations, and best practices for scheduling applications built with Blazor and ASP.NET Core technologies.

## Project Overview

The Blazor Scheduler is a powerful scheduling component that enables developers to create feature-rich calendar applications. This repository provides practical examples of working with different appointment options, including recurring events, time slots, resource management, and appointment customization. It serves as a reference implementation for developers looking to integrate advanced scheduling functionality into their Blazor applications.

## Features

* **Appointment Management**: Create, edit, and delete appointments with full control
* **Recurring Appointments**: Support for recurring events with various recurrence patterns
* **Resource Scheduling**: Manage appointments across multiple resources
* **Time Slot Configuration**: Customize working hours and business hours
* **Appointment Customization**: Personalize appointment appearance and behavior
* **Real-time Updates**: Dynamic appointment management with immediate UI reflection

## Prerequisites

* Visual Studio 2022
* .NET 6.0 or later
* Syncfusion Blazor License or Community License

## Getting Started

### Installation

1. Checkout this project to a location on your disk
2. Open the solution file using Visual Studio 2022
3. Restore the NuGet packages by rebuilding the solution
4. Run the project

### Configuration

Configure the Scheduler component in your Blazor pages to enable appointment management:

* Set up appointment data sources
* Configure working hours and business hours
* Define resource settings
* Customize appointment templates
